# 劇本寫作管理系統

一個專為劇本創作者設計的寫作進度管理工具，幫助你追蹤寫作進度、管理劇本大綱和場次內容。

## 功能特色

### 1. 寫作進度總覽 (Dashboard)
- **場次總數顯示**：追蹤你的劇本總場次數
- **剩餘天數顯示**：根據設定的完結日期自動計算
- **每日需完成場次**：自動計算每天需要完成的場次數
- **進度條**：視覺化顯示整體寫作進度
- **完結日期設定**：可自由選擇目標完成日期

### 2. 劇本大綱
- 採用15節拍點結構，幫助你建立完整的劇本架構
- 每個節拍點都有說明，引導你思考故事發展
- 支援大綱內容的即時編輯和儲存

### 3. 場次管理
- **場次列表**：以卡片形式展示所有場次
- **場次編輯**：點擊場次即可進入編輯模式
- **Markdown支援**：使用Markdown語法撰寫場次內容
- **完成狀態**：可標記場次為已完成
- **新增/刪除**：輕鬆管理場次

### 4. Markdown編輯器
- 完整的Markdown語法支援
- 即時預覽功能
- 工具列包含常用格式選項
- 自動儲存功能

## 使用方式

### 啟動應用

1. 進入「劇本管理平台」目錄：
```bash
cd ~/Desktop/劇本管理平台
```

2. 安裝依賴（如果尚未安裝）：
```bash
npm install
```

3. 啟動開發伺服器：
```bash
npm run dev
```

4. 在瀏覽器中開啟：
   - 訪問 `http://localhost:3001` 進入劇本管理平台

### 基本操作

1. **設定寫作目標**
   - 在Dashboard中設定場次總數
   - 選擇完結日期
   - 系統會自動計算每日需完成的場次

2. **建立劇本大綱**
   - 在「劇本大綱」區塊中，依序填寫15個節拍點
   - 每個節拍點都有說明，幫助你思考故事結構

3. **新增場次**
   - 點擊「新增場次」按鈕
   - 場次會自動編號

4. **編輯場次**
   - 點擊場次卡片進入編輯模式
   - 使用Markdown語法撰寫內容
   - 內容會自動儲存

5. **標記完成**
   - 勾選場次前的核取方塊
   - 進度會自動更新

## 資料儲存

所有資料都儲存在瀏覽器的本地儲存（localStorage）中，包括：
- 寫作進度資料
- 劇本大綱
- 所有場次內容

**注意**：清除瀏覽器資料會導致所有內容遺失，建議定期備份重要內容。

## 技術架構

- **前端框架**：React 18
- **建置工具**：Vite
- **路由**：React Router
- **Markdown編輯器**：EasyMDE
- **樣式**：CSS3

## 瀏覽器支援

支援所有現代瀏覽器：
- Chrome (推薦)
- Firefox
- Safari
- Edge

## 注意事項

- 本應用使用本地儲存，資料僅存在於你的瀏覽器中
- 建議定期備份重要內容
